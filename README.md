# Baby Phish

A location and camera phishing tool for Linux.

## Requirements

- Linux OS
- PHP
- wget
- unzip

## Installation

```bash
sudo apt install php wget unzip
chmod +x babyphish.sh
```

## Usage

```bash
./babyphish.sh
```

## Templates

1. Online Meeting (Zoom)
2. Telegram Premium
3. Porn Site (FUQ)
4. WhatsApp Group

## Tunnel Options

1. CloudFlare Tunnel
2. Loophole

## Features

- GPS location tracking
- Video capture
- Multiple phishing templates
- Cloudflare and Loophole tunneling support
