# Baby Phish

A location and camera phishing tool for Linux.

## Requirements

- Linux OS
- PHP
- wget
- unzip

## Installation

```bash
sudo apt install php wget unzip
chmod +x babyphish.sh
```

## Usage

```bash
./babyphish.sh
```

## Templates

1. Telegram Channel
2. WhatsApp Group

## Tunnel Options

1. CloudFlare Tunnel
2. Loophole

## Features

- GPS location tracking
- Camera & Video capture
- Telegram Bot Alerts (Real-time notifications)
- Cloudflare and Loophole tunneling support
